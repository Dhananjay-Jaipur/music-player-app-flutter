# Spotify Clone in Flutter 

A working Spotify clone built in Flutter using Firebase.

## Songs and Artists available on app
 * Arjit Singh
    * Agar Tum Saath Ho
  * Iqlipse Nova
    * Mera Safar
    * Aarzoo
  * Imagine Dragons
